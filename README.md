# Instagram-User-Activity-Demographics-Analysis
This project focuses on analyzing Instagram user lifestyle and engagement data using Python. The dataset was cleaned and prepared by handling missing values and removing duplicates to ensure accuracy. Exploratory data analysis was performed to understand user demographics such as age, gender, and country. The project also examined the relationship between lifestyle factors like sleep, exercise, and perceived stress levels.An interactive Power BI dashboard was developed to present insights in a clear and user-friendly way. The cleaned data was stored in a MySQL database using SQLAlchemy. This project demonstrates end-to-end data analysis and visualization skills applicable to real-world business problems.
